name - имя проекта. Не должно совпадать с пакетами, которые вы устанавливаете
dependencies - те библиотеки, которые должны попасть в конечный код
devDependencies - библиотеки для удобства написания кода
scripts - (
    Модули, которые мы устанавливаем, могут использоваться в объекте scripts
    (В терминале они недоступны)

    Структура:
    scripts: {
        "Название": "Сам код, который необходимо выполнить"
    }
)


Нужно для модулей, которые загружаются на npmjs:
version - версия. 
description - Описание проекта
author - автор
license - лицензия
